# 🎬 Cinematic Dual Portfolio

A stunning, Netflix-inspired dual portfolio website built with modern web technologies. This project features a sleek, cinematic user interface complete with an ambient GPU-accelerated background, smooth transitions, and a premium dark-mode aesthetic.

## ✨ Features

- **📺 Netflix-Style UI:** Immersive, visually striking interface inspired by modern streaming platforms.
- **🎭 Dual Profiles:** Support for presenting two distinct portfolios (or profiles) seamlessly within a single application.
- **🌌 Cinematic Ambient Background:** A sophisticated, GPU-accelerated blurred orb background that dynamically enhances the visual experience.
- **⚡ Ultra-Fast Performance:** Built on **Next.js 14** (App Router) and optimized for rapid page loads and fluid navigation.
- **🎨 Beautifully Styled:** Hand-crafted designs using **Tailwind CSS v4** with a focus on dark mode, glassmorphism, and micro-animations.
- **📱 Fully Responsive:** Flawless experience across desktops, tablets, and mobile devices.

## 🛠️ Tech Stack

- **Framework:** [Next.js](https://nextjs.org/) (React 19)
- **Language:** TypeScript
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Deployment:** Vercel (Recommended)

## 🚀 Getting Started

Follow these instructions to get a copy of the project up and running on your local machine.

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed along with `npm`, `yarn`, `pnpm`, or `bun`.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Lakshya-M/LM-Portfolio-.git
   cd LM-Portfolio-/my-portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   # or yarn install, pnpm install, bun install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   # or yarn dev, pnpm dev, bun dev
   ```

4. **Open in browser:**
   Navigate to [http://localhost:3000](http://localhost:3000) to view your cinematic portfolio.

## 💡 How It Works

The main layout wraps the entire application in a clean structure containing navigation headers, main content regions, and a customized futuristic footer. The application utilizes modern React patterns to smoothly transition between different views and showcases two contrasting profile sets dynamically. The cinematic background is purely CSS-driven and does not conflict with performance.
